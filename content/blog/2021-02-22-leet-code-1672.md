# Richest Customer Wealth

~~~java
class Solution {
    public int maximumWealth(int[][] accounts) {
        return Arrays.stream(accounts).map(arr -> Arrays.stream(arr).sum())
          .max(Integer::compareTo)
          .get();
    }
}
~~~
